# Summary

Native Android plugin for Unity. Provides OpenXR frame metrics, notably the last frame delivery time interval.

# Build

Run `build.ps1` in PowerShell. Then copy the resulting .so file from the "build-android" folder to the Android plugins folder in your Unity project.

# Details

OpenXR C headers were downloaded from here:  
https://github.com/KhronosGroup/OpenXR-SDK/tree/main/include/openxr
